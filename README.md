# angular-hybrid-sample

This is a sample of angular hybrid app (angular8 and angularjs) with a sample of using angularJS directives and factories

### Installation

```
npm install
```

### Start Dev Server

```
npm start
```

### Build Prod Version

```
npm run build
```

## Tech

- UI-Router angular-hybrid 
- Angular 8.1.2
- AngularJS 1.6.6
- Typescript 3.4.0
